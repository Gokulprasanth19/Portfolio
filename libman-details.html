Library Management System Overview

This web-based Library Management System allows users to register, log in, and manage books through a responsive dashboard. Built using React for a dynamic UI and Bootstrap for styling, it delivers a seamless user experience.

  • User registration and login with form validation
  • Error feedback for incorrect credentials
  • Interactive dashboard displaying book stats
  • Add, edit, and delete books with real-time updates
  • Reusable components and clean React structure

This project was developed during an internship and reflects real-world design practices and user interface development with React.
